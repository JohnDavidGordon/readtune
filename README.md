# readtune
Personalised readability calibration tool.

# Thanks for being here!

ReadTune is a personalised readability calibration tool.

Rather than assuming which font, spacing, colours, or layout work best for a reader, ReadTune learns individual preferences through a series of adaptive A B testing.

Users are shown two versions of the same text and select the version they find easier to read.

Over time, ReadTune builds a personal readability profile and provides practical recommendations applicable across websites, documents, e-readers, and other digital content.

## Goals

- Personalised rather than generic
- Evidence-based rather than assumption-based
- Fast and intuitive to use
- Accessible to users with and without dyslexia
- Produce a useful take-away profile

## Version 0.1

- Adaptive A/B comparisons
- Typography and colour optimisation
- Confidence scoring
- Personal readability profile
- Local-first architecture

## Future Versions

- Bayesian optimisation
- Extended readability profiles
- Browser integrations
- Document reformatting
- Accessibility research tools

## Core Principle

The easiest text to read is not necessarily the same for every reader.
ReadTune aims to discover what works best for each individual.
